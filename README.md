Tugas ini dibuat oleh **Muhammad Affandes** (affandes@uin-suska.ac.id)

## Instruksi

Buatlah sebuah class dengan nama `Mahasiswa` yang memiliki field sebagai berikut:

1. Field `nama`
2. Field `nim`
3. Field `umur`

Tentukan masing-masing tipe data yang sesuai untuk masing-masing field.

Setelah class `Mahasiswa` dibuat, buatlah sebuah object dari class tersebut pada `psvm` yang ada di class `Main`. Setiap
field pada object tersebut (`nama`, `nim`, `umur`) diinputkan menggunakan `Scanner` atau `BufferedReader`. Terakhir,
outputkan nama, nim dan umur yang dari object yang telah dibuat tadi.

## Input

```shell
Andi
100012
19
```

## Output

```shell
Andi 100012 (19 tahun)
```

## Direktori

Gunakan direktori berikut ini untuk membuat file class pada tugas ini:

```shell
/src/main/java/com/github/affandes/kuliah/pl/pl01
```

Apabila direktori tersebut tidak tersedia, silahkan dibuat sesuai dengan alamat direktori tersebut.
